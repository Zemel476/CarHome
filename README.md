# CarHome
汽车之家二手车数据爬取

## 关于本项目
本项目是我个人用于学习和实践练习项目。项目中的代码、架构或实现方式可能并非最佳实践，也可能存在未完善的环节，主要目的在于记录学习过程和验证想法。

主要爬取汽车之家二手车车辆档案和详细配置数据，并存储在csv文件中，文件路径在 setting 可进行配置。
车辆档案和详细配置信息仅抓取部分作为代码示例。
